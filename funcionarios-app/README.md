# Sistema de Gestión de Funcionarios - PostgreSQL

Aplicación web completa para la gestión de funcionarios, cargos, proyectos y asistencia laboral, utilizando PostgreSQL como base de datos.

## Características

- ✅ CRUD completo para todas las entidades
- ✅ Dashboard con estadísticas
- ✅ Control de asistencia
- ✅ Gestión de vacaciones
- ✅ Reportes y estadísticas
- ✅ Responsive design
- ✅ Modo demo para GitHub Pages
- ✅ Exportación de datos
